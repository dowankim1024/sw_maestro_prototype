# 이슈캐스트 — 제품 구조 정리

> 피드백 반영 후 정리본. 기획 문서 v1을 대체하지 않고, **방향성 결정 사항**만 압축.

---

## 0. 한 줄 정의

**같은 이슈를 4명의 시각으로 30초 안에 짚어보고, 더 궁금하면 그 시각의 캐릭터와 가볍게 대화하는 모바일 이슈 앱.**

---

## 1. 핵심 결정: 누구를 위한 제품인가

### 우선순위

| 순위 | 사용자 | 역할 |
|---|---|---|
| 1순위 (Wedge) | **정보 깊이 추구자** | 제품 정체성을 만드는 코어. 4명의 시각·대화·"오늘 새로 본 것" 누적이 이 사용자한테 작동해야 제품이 산다. |
| 2순위 (Funnel) | **Talking Point 추구자** | 진입 트래픽. 카드 30초만 보고 닫아도 가치가 있어야 하고, 그중 일부가 깊이 추구자로 전환되는 입구 역할. |

### 왜 깊이가 wedge인가

- **Talking Point 시장은 이미 포화** — 뉴닉, 어피티, 머니레터, 카카오 뷰가 다 잘함. "더 빠르고 트렌디"만으로는 차별화 약함.
- **정보 깊이 시장은 비어 있음** — 신문/유튜브/팟캐스트는 무겁고, 뉴닉/어피티는 얕음. 그 사이 자리(가벼운 깊이)가 비어 있음.
- **"4명의 시각 + 캐릭터 대화"는 카테고리적으로 비어 있는 자리** — character.ai 식 캐릭터 매력 + 이슈 큐레이션의 결합은 새로움.

### 두 사용자가 같은 제품에서 만족하는 방식

```
[Talking Point 추구자]
  카드 30초 → "아 그거 알아" → 닫음
  성공 = "오늘도 하나 알아갔네"

[정보 깊이 추구자]
  카드 → 4명 시각 비교 → 캐릭터 대화 → 핸드오프 → "오늘 새로 본 것" 저장
  성공 = "이 사안을 이전보다 잘 이해했다"
```

## 2. 제품 원칙

### 비논쟁 노선 (Non-Debate)

- 토론·찬반·점수·등급·레벨·자유 댓글 모두 **스펙아웃**
- "토론" 단어 → "대화" 또는 "시각"으로 통일
- 캐릭터 간에도 서로 디스하지 않음. "그 사람은 그 영역을 더 잘 안다"로 핸드오프
- 사용자 반응은 이모지/짧은 공감만, 자유 텍스트 댓글 없음

### 두 시간 단위

- **30초** = 카드 (Talking Point 추구자의 종착점, 깊이 추구자의 진입점)
- **N분** = 4명의 시각 + 대화 (깊이 추구자의 영역)
- 카피 예시: "30초로 시작, 더 보면 N분"

### 환각·오정보 대응

이건 MVP 전 필수. 면책 한 줄로 끝낼 게 아니라 **인터페이스 차원에서 강하게**.

- **출처 표기 강제** — 핵심 사실 카드의 모든 항목에 출처 라벨
- **팩트 vs 의견 구분 UI** — `팩트`/`보도됨`/`분석` 같은 라벨로 시각적으로 분리
- **민감 이슈 동적 면책** — 모든 이슈에 똑같은 면책 띄우지 말고, 정책·사회 이슈 같은 민감 카테고리에만 추가 면책 박스
- **캐릭터 발언 = AI 생성 의견 명시** — 카드와 대화 양쪽에 표시
- **대화 중 가드레일** — 캐릭터가 모르면 "모른다" 말하기, 새로운 통계·숫자 만들어내지 않기, 핵심 사실 카드로 돌려보내기

### 모바일 우선

- 카드/이미지 중심, 긴 문장 지양
- 좌우 스와이프 = 다른 시각, 상하 스와이프 = 다른 이슈
- 데스크톱은 대응만, 최적화는 모바일

## 3. 4가지 시각 (렌즈)

같은 이슈를 4개 렌즈로 본다. **정치·세대·젠더 축은 안 건드림.** 충돌이 아니라 **다른 정보 레이어**를 쌓는 구조.

| 렌즈 | 핵심 질문 | 강조하는 정보 |
|---|---|---|
| **생활자** | 내 일상에 뭐가 닿나? | 체감, 가격, 직접 경험 |
| **전문가** | 구조적으로 어떻게 작동하나? | 배경, 원리, 비교 사례 |
| **트렌드** | 지금 사람들이 어떻게 반응하나? | 현장, 밈, 문화 현상 |
| **회의주의자** | 이거 진짜야? 다른 면은? | 검증, 출처, 다른 가능성 |

### 왜 "거시" 대신 "회의주의자"인가

- 거시 렌즈는 시사 이슈에선 강하지만 **트렌드성 이슈(빅뱅 컴백, 러닝 열풍, 신메뉴 유행 등)에서 어색함** — "정책적 의미는…"이 무리한 톤이 됨.
- 회의주의자 렌즈는 **모든 이슈 타입에서 자연스럽게 작동** — "이거 진짜야?"는 시사든 트렌드든 통함.
- wedge에 가장 강하게 기여 — 다른 세 캐릭터가 이슈를 수용하는 톤일 때 한 명이 "잠깐"을 던져 다층성을 만듦.
- 비논쟁 노선과 충돌 안 함 — 토론 거는 게 아니라 **다른 각도의 정보를 추가**하는 거.

## 4. 화면 구조

### 이슈 상세 화면

```
[제목 + 한 줄 요약]                         ← Talking Point 추구자 종착점 후보
[민감 이슈일 때 동적 면책 박스]
[핵심 사실 3개 + 출처 라벨]                  ← 둘 다 필수로 봄
[왜 지금 뜨는지]
─────────────────────────────────────
[같은 이슈, 4명의 시각]                      ← 깊이 추구자 진입점
"한 명만 빠르게 봐도 OK" 안내                ← 부담 빼는 카피 (잘 작성됨)
[캐릭터 카드 4개 — 좌우 스와이프]
─────────────────────────────────────
[캐릭터 클릭 시 → 대화 진입]
```

### 핵심 디자인 결정

- **사실이 먼저, 시각이 그다음** — 순서 절대 바뀌면 안 됨. 캐릭터 의견이 사실처럼 읽히는 거 방지.
- **각 캐릭터 발언 옆에 "이 시각이 닿은 핵심 사실 #N" 라벨** — 환각 리스크 대응. 사용자가 사실과 시각의 관계를 시각적으로 잡을 수 있음.
- **민감 이슈 면책은 동적으로** — 모든 이슈에 띄우면 면책 피로로 안 읽힘. 정책·사회 카테고리에만.

## 5. 대화 시스템 (깊이 추구자용)

### 대화의 정체성

**아닌 것**:
- ❌ 토론 (찬반 부딪치기) — 비논쟁 노선 어김
- ❌ 일반 챗봇 (뭐든 답하는 ChatGPT) — wedge 사라짐
- ❌ 검색 (정보 더 캐기) — 뉴닉/구글이 더 잘함

**맞는 것**:
- ✅ 이 시각을 가진 사람과 잠깐 더 얘기해보는 것
- ✅ 캐릭터의 **렌즈 안에 머무는** 대화
- ✅ 출구가 명확한, 잠깐 들렀다 가는 자리

### 대화를 끌어가는 4가지 패턴

**1. 추천 질문 칩** — 캐릭터별 톤 다르게
- 캐릭터 발언 아래 칩 3개 노출
- 타이핑 부담 제거 (30초 컨셉과 일치)
- 캐릭터별로 묻는 톤이 다름 (생활자한텐 친근하게, 전문가한텐 정중하게)

**2. 캐릭터의 후속 미끼** — 자기 렌즈로 사용자 경험 묻기
- "너는 자취해? 월세 얼마 내?" (생활자)
- "이 정책 직접 받아본 적 있어요?" (전문가)
- 의견 질문(❌)이 아니라 경험 질문(✅) — 토론으로 안 가면서 깊은 대화로 연결

**3. 다른 캐릭터로 핸드오프**
- 캐릭터가 자기 영역 밖이면 다른 캐릭터에게 양보
- "그건 내 영역 아니야. 수진 교수님이 잘 알지" → [수진 교수와 이어서 얘기하기] 버튼
- "같은 이슈 N개 시각" wedge를 대화 중에도 살리는 핵심 동선
- 캐릭터들이 디스 안 함 — "그 사람이 더 잘 안다"

**4. 대화 마무리 = "오늘 새로 본 것" 카드 자동 생성**
- 비논쟁 노선의 거의 유일한 보상 경험
- 대화에서 새로 본 시각을 카드로 뽑아 누적
- "내 카드" 메뉴에 컬렉션으로 쌓이는 게 리텐션 동력

### 대화 길이

- 30초 컨셉과 안 부딪치는 설계 필요
- 3-4 턴쯤에 자연스러운 마무리 신호 ("이 정도가 내가 줄 수 있는 시각이야")
- 매 턴마다 자연스러운 출구 보임: [다른 시각도 보기] [오늘 새로 본 것 저장]
- character.ai 식 끝없는 대화는 안 어울림

## 6. 리텐션 전략

### 매일

- **새 이슈 업데이트** (콘텐츠 신선도)
- **푸시 / 위젯**: "오늘의 이슈 도착" (가벼움) + "○○이 오늘 이슈에 의견 남김" (깊이)

### 누적

- **"오늘 새로 본 것" 컬렉션** — 대화 종료 시 자동 생성, "내 카드"에 누적
  - 비논쟁 노선의 핵심 보상
  - 시간 흐를수록 가치 누적되는 자산
- **같은 이슈를 다른 캐릭터로 다시 보기** — 한 이슈에 N번 재방문 동기

### KPI 우선순위

깊이 추구자 중심이니까 카드 조회수보다:
- **대화 진입율** — 카드 본 사람 중 캐릭터 대화로 들어가는 비율
- **"오늘 새로 본 것" 저장 수** — 깊이 추구자가 가치를 느낀 증거
- **재방문 빈도** — 매일/주간 컬렉션 누적 패턴

## 7. 의도적으로 빼는 것 (스펙아웃)

| 항목 | 이유 |
|---|---|
| 찬반 토론, AI 토론 모드 | 비논쟁 노선과 충돌, 정체성 흐트러짐 |
| 레이팅·점수·레벨 (지식인 태양신 류) | 토론 점수화는 비논쟁 노선과 충돌 |
| 자유 텍스트 댓글 | 사람-사람 댓글은 자연스럽게 논쟁으로 — 반응(이모지)만 |
| "토론" 용어 | 전부 "대화" 또는 "시각"으로 |
| 거시 렌즈 (4번째 캐릭터) | 트렌드성 이슈에서 어색 — 회의주의자로 대체 |

## 8. MVP 전 반드시 해결할 것

1. **출처 표기 자동화 파이프라인** — AI가 임의로 출처 만들어내지 않게
2. **팩트 vs 의견 구분 UI** — 라벨 시스템 디자인 확정
3. **민감 이슈 자동 분류 + 동적 면책** — 어떤 카테고리/키워드에서 면책 박스 띄울지
4. **캐릭터 톤 일관성 검증 시스템** — LLM이 흔들릴 때 가드레일
5. **"오늘 새로 본 것" 카드 자동 생성 프롬프트** — 대화에서 핵심 인사이트 추출
6. **벤치마킹 — 뉴닉, 어피티 깊게 사용** — 30초 카드 컨벤션, 톤, 카피 학습

## 9. 향후 고민거리 (지금은 결정 안 함)

- 스트릭/연속 방문 같은 동기부여 시스템 (도입 시 비논쟁 노선과의 균형 검토 필요)
- 사용자 간 카드 공유 기능 (논쟁 유발 위험 평가 필요)
- 캐릭터 추가 (5명, 6명 — 단 wedge 흐트러지지 않는 선에서)
- 수익 모델 (구독, 광고, B2B 라이선싱 등 별도 검토)
